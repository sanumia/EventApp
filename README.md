EventApp
EventApp — это серверное приложение (бэкенд) для создания и управления мероприятиями, разработанное на платформе .NET. 
Проект демонстрирует применение принципов чистой архитектуры и многослойной (N-Layered) структуры для создания масштабируемого, поддерживаемого и тестируемого кода.

Основные характеристики
Архитектура: Приложение построено по многослойной архитектуре с четким разделением ответственности между слоями (Domain, DataAccess, Application, API). Это обеспечивает низкую связанность, высокую связность и упрощает юнит-тестирование.

Технологический стек: Используется современный стек разработки под .NET: C#, ASP.NET Core, Entity Framework Core (PostgreSQL), AutoMapper.

Слой данных: Для работы с базой данных реализован шаблон Repository, что абстрагирует логику доступа к данным и упрощает смену persistence-хранилища. Для связи с БД используется Entity Framework Core (Code-First подход).

Серверная часть: Основная бизнес-логика инкапсулирована в сервисах слоя Application. Для безопасной передачи данных между слоями используются DTO (Data Transfer Objects).

API: Внешним интерфейсом приложения служит RESTful API, реализованный с помощью контроллеров ASP.NET Core.
